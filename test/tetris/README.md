# HSP3Dish Tetris

HSP3Dish validation sample for the `hsp-script` skill.

## Run

Open `tetris.hsp` from the HSP 3.7 Script Editor, or compile it with an HSP3Dish runtime setup that can resolve `common/hsp3dish.as`.

## Controls

- `A` / `D` or left / right: move left / right
- `W` or up: rotate
- `S` or down: soft drop
- `Space`: hard drop
- `R`: restart after game over
